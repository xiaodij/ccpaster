How to use: Paste Claude Code output on the left. The right panel shows the formatted result. Click Copy for Email to copy rich text you can paste directly into Gmail, Outlook, etc. Click Copy HTML to get the raw HTML source.
Supported formats: Box-drawing tables (Unicode), Markdown pipe tables, headers (#), bold/italic (**/_), code blocks (```), inline code (`), bullet/numbered lists, blockquotes (>), horizontal rules (---), and ANSI escape codes (stripped).
